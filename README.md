# 💳 Fraud Detection System

An end-to-end application that detects fraudulent transactions from bank statements.

## 🚀 Features
- Upload bank statements (PDF/CSV)
- Extract transaction data automatically
- Fraud detection using Isolation Forest
- Category classification (Food, Shopping, etc.)
- Interactive dashboard
- Chatbot for querying insights
- SQLite database integration

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
- PDFPlumber

## ▶️ Run Locally
pip install -r requirements.txt
streamlit run app.py
